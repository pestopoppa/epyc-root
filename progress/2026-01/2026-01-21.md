### REPL Document Functions

| Function | Description |

| `section(n)` | Get section n content (1-indexed) |
| `figures(section=n)` | List figures, optionally filtered by section |

| `document_info()` | Get document metadata |

### Document Detection

The `DocumentPreprocessor.needs_preprocessing()` detects documents via:

1. **File extension:** `.pdf`, `.png`, `.jpg`, `.jpeg`, `.tiff`, `.bmp`, `.docx`
2. **MIME type:** `application/pdf`, `image/png`, `image/jpeg`, etc.
3. **Task type:** `doc`, `ingest`, `document`
4. **Explicit config:** `preprocessing.ocr.enabled: true`
5. **Prompt phrases:** "ocr", "scan", "extract text", "read pdf"

### Data Classes

```python
@dataclass
class BoundingBox:
    id: int
    x1, y1, x2, y2: int  # Normalized [0, 1000]

@dataclass
class Section:
    id: str              # "s0", "s1", etc.
    title: str           # Header text
    level: int           # 1, 2, or 3
    content: str         # Section body
    page_start, page_end: int
    figure_ids: list[str]

@dataclass
class FigureRef:
    id: str              # "p1_fig0"
    page: int
    bbox: BoundingBox
    description: str
    section_id: str | None

@dataclass
class DocumentPreprocessResult:
    original_path: str
    sections: list[Section]
    figures: list[FigureRef]
    total_pages: int
    failed_pages: list[int]
    processing_time: float
```

### Usage Example

```python
from src.services.document_preprocessor import preprocess_documents

# Preprocess a TaskIR with document inputs
task_ir = {
    "inputs": [{"type": "path", "value": "/path/to/doc.pdf"}]
}

result = await preprocess_documents(task_ir)
if result.success:
    doc = result.document_result

    # Access sections
    for section in doc.sections:
        print(f"{section.title}: {len(section.content)} chars")

    # Access figures
    for fig in doc.figures:
        print(f"Figure {fig.id} on page {fig.page}")

# Or use REPL environment
from src.repl_document import create_document_repl

repl = create_document_repl(doc)
result = repl.execute("print(sections())")
result = repl.execute("print(section(1))")
result = repl.execute("print(figures())")
```

### Forward Pass Design
1. Main loop iterates `n_layer - nextn_predict_layers` (excludes MTP)
2. Saves initial embedding + last hidden state
3. MTP: `enorm(embd)` + `hnorm(hidden)` → concat → `eh_proj` → gated attention (no-cache via `build_attn_inp_no_cache`) + MoE → `shared_head_norm` → shared LM head
4. MTP logits stored in `res->t_logits_mtp`
5. Build succeeds, backward compatible (nextn_predict_layers=0 default)

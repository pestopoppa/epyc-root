## Corpus Index V3: Sharded Build + Pruning

### Summary
Rebuilt corpus index pipeline from scratch after previous day's failed phased approach.
Deleted JS, TS, Go corpora (not needed). Pruning remaining 3 languages by 50% via
deterministic sampling during population into shards.

### Decisions
- **Dropped languages**: JavaScript, TypeScript, Go — not needed for current use cases
- **50% sampling**: `id % 1000 < 500` — uniform across all repos, preserves diversity
- **Dual-cursor merge**: Fixed O(N²) bug in original populate script (per-snippet ngram

## claude-mem Plugin: 637 Zombie Workers (231 GB RAM Leak)

### Problem
System RAM at 818 GB with 7.5 GB swap consumed despite a healthy idle orchestration
stack (~570 GB expected). Investigation found **637 orphaned Claude subprocess workers**
(`claude.*stream-json`) spawned by the claude-mem plugin's bun worker daemon (PID 2985064,
`worker-service.cjs`). Workers spawned between Feb 16 19:09 and Feb 17 01:02, consuming
**228 GB RSS + 2.6 GB swap = ~231 GB total**. Additionally, 6 zombie processes found
(5 curl from NextPLAID health checks, 1 speech-dispatcher) — harmless, zero resources.

### Root Cause
claude-mem v9.0.12 had no subprocess concurrency limit. The `SDKAgent.startSession()`
spawned new Claude SDK agent subprocesses without bound. During a heavy session with
many observations queued, the worker kept spawning new agents without reaping old ones.

### Fix
- **Plugin update**: `~/.claude/plugins/marketplaces/thedotmack/` pulled from
  `1341e93` (v9.0.12) → `eea4f59` (v10.2.3). Key upstream fixes:
  - `51719d2` — `CLAUDE_MEM_MAX_CONCURRENT_AGENTS` setting (default: 2),
    `waitForSlot()` in ProcessRegistry with 60s timeout
  - `cd31eaf` — Parent heartbeat for MCP server orphan prevention
  - `a3f9e7f` — 5-layer spawn storm defense ("641 processes → max 2")
- **Cleanup**: Killed 637 workers (SIGTERM, all exited cleanly), restarted bun daemon
- **Result**: RAM 818→577 GB, swap 7.5→0 GB, workers 637→2

### Files / Actions
| Action | Detail |
|--------|--------|
| `git pull origin main` | `~/.claude/plugins/marketplaces/thedotmack/` (v9.0.12 → v10.2.3) |

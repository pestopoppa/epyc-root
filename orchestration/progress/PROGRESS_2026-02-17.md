# Orchestration Progress: 2026-02-17

## Deprecate `coder_primary` Role

### Summary

Hard-removed the `coder_primary` role from the entire codebase. `coder_primary` and `frontdoor` were the same model (Qwen3-Coder-30B-A3B) on the same port (8080) — escalating from frontdoor to coder_primary was self-delegation. Code tasks now route directly to `coder_escalation` (Qwen2.5-Coder-32B, port 8081).

### Root Cause: Instruction Stacking vs Few-Shot Learning

The `rules.md` prompt had 51 lines of instructions about WHEN and HOW to use FINAL(). Models don't learn protocols from instruction lists â€” they learn from examples. The model echoed template placeholders (`FINAL("answer")`) because it saw `FINAL(answer)` in the rules and copied it literally.

### Fixes Applied

| Fix | File(s) | Impact |
|-----|---------|--------|
| Few-shot prompt rewrite | `rules.md`, `root_lm_system.md`, `builder.py` | 8 concrete examples replace 51 rules. Teaches protocol by demonstration. |
| Wasteful delegation guard | `chat_delegation.py` | Blocks architectâ†’coder delegation for non-code questions with numeric briefs |
| `wasteful_delegation` signal | `anomaly.py` | Detects short/numeric answers after delegation (weight 0.5) |
| `repl_max_turns` signal | `anomaly.py` | Detects max-turns exhaustion without FINAL() (weight 1.0) |
| Late-game FINAL() nudge | `nodes.py` | DEADLINE injected at â‰¤3 remaining turns |
| Status-phrase expansion | `anomaly.py`, `nodes.py` | "code", "explanation of code or reasoning", etc. added to rejection/detection sets |
| Template echo prevention | `tools.md`, `rules.md` | `FINAL(answer)` â†’ `FINAL(value)` to reduce literal echoing |

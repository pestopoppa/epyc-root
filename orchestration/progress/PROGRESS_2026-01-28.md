# Orchestration Progress: 2026-01-28

## VL Model Selection Research

### Summary

Comprehensive VL model evaluation for orchestration document analysis pipeline. Tested all VL models on Twyne whitepaper figures with and without document context.

### Key Finding: Smaller Models Outperform Larger

| Model | VL Score | Speed | Result |
|-------|----------|-------|--------|
| **Qwen3-VL-4B** | **94%** | 18 t/s | **BEST** |
| Qwen3-VL-8B | 86% | 15 t/s | Good alternative |
| Qwen2.5-VL-7B | 81% | 17 t/s | Only agentic option |
| Qwen3-VL-30B-A3B | 75% | 27 t/s | OCR errors |
| Qwen3-VL-235B | 56% | 4.6 t/s | Timeout truncation |

### Why 4B Beats 30B/235B

1. No timeout truncation (completes within limits)
2. Accurate OCR - correctly reads chart labels
3. Chart legend interpretation is accurate
4. Smaller = faster context processing

### Document Context Research

Tested figure analysis with varying context levels:

| Context | Prompt Tokens | Speed | Quality |
|---------|---------------|-------|---------|
| None | ~500 | 18 t/s | Describes in isolation |
| Summary (~8K) | 5,014 | 13.8 t/s | **Near-optimal quality** |
| Full OCR (34K) | 12,342 | 7.4 t/s | Marginal improvement |

**Conclusion**: Summary context provides ~90% of full OCR benefit at 2x speed.

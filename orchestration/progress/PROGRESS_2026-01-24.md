### Architecture Highlights

**Token Efficiency**: ~350 tokens per operation vs 3000-5000 for manual execution

**Safety Features**:
- `deprecate_model` requires admin role, approval, and generates manual delete command (NEVER auto-deletes)
- All registry modifications create patches for owner approval
- Rollback support with checkpointing

**Three-Layer Design**:
```
REPL Environment (tools)
    ↓
Procedure Registry (YAML definitions)
    ↓
Episodic Memory (learns from outcomes)
```

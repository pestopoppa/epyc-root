## Changes Made

### 1. Lookup Test Speed-Only Configuration

**Problem:** Lookup tests were running full quality suites (9 questions × 3 n-gram values = 27 runs) when quality doesn't change from baseline.

**Solution:** Added `speed_test_only = True` and `inherits_quality_from` to lookup configs.

| File | Change |
|------|--------|
| `scripts/lib/executor.py` | Added speed_test_only flags to lookup configs (lines 557-562, 609-616) |
| `scripts/benchmark/run_benchmark.py` | Added LOOKUP_SPEED_TEST_PROMPT (~1K tokens for n-gram matching) |

**Result:** ~90% reduction in lookup benchmark time.
